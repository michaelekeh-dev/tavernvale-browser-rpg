# Tavernvale Browser RPG

A browser-based RPG prototype built as an experimental game project. The game includes progression systems, world events, combat-related gameplay, UI overlays, audio, and shop/economy features and lots more!!!

## Features

- Browser-based RPG gameplay
- UI overlays and interface systems
- Audio integration
- Shop and economy systems
- World event / boss event features
- Admin/testing pages
- Ongoing feature iteration and balancing

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js / npm 
- Docker / deployment config
- Railway (used for website hosting)

## Project Structure

- `rpg.html` - main RPG/game page
- `player.html` - player-related interface
- `admin.html` - admin/testing functionality
- `game.js` - main gameplay logic
- `Overlay/` - overlay/UI-related assets
- `audio/` - music and sound assets
